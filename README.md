Welcome to # mungeondaster.github.io

This web interface was developed to easily track and maintain warband rosters for the tabletop game Mordheim.

Mordheim is a classic 1999 tabletop skirmish game by Games Workshop set in the Warhammer Fantasy universe. Players control small, customized warbands battling through a ruined, chaos-infested city to scavenge "wyrdstone". It is highly celebrated for its deep narrative campaigns, permadeath mechanics, and RPG-style character progression.

# How to use:
The web app allows you to "Muster a Warband" and select from the 8 original warbands (1a).

# Campaigns
The app was built to allows friends to play in the same campaign together and view eachother's warband rosters.  Click the "Campaign" button and give it a name.  Your campaign is then assigned a unique code that can be easily shared with friends.  *Note: warbands are currently editable by any user in the campaign.  Keep your hands to yourself.
